# IBANDL-Archive (2012-2026)
Ion Beam Analysis Data Library (IBANDL)\
Archive created by V.Zerkin, IAEA-NDS, 2023

### Storage of R33 files
1) File extensions are ".r33" (files with "names.R33" are renamed to "names.r33")
2) File names are in lower-case,
   e.g. "n5aa0Wilmes1.r33" is renamed to "n5aa0wilmes1.r33"
3) Using "\\n" as end-of-line (Unix-style); "\\r\\n" are replaces by "\\n" in order to \
   minimize diference in the file-history for the files edited in different OS
